## 包含的组件（推测）

> 基于 React UI 组件库的常见结构

- **Button**（带 LED 状态灯 + pulse 动画）
- **Card**（带边框 + 阴影 + 扫描线 overlay）
- **Modal**（带 CRT 边框）
- **Input**（终端风）
- **Tabs**（带 Led 切换动画）
- **Table**（终端 mono 字体）
- **Chart**（复古仪表）
- **Tooltip**（霓虹光）

## 怎么用这个项目学习

### 用法 1：直接当组件库用

```bash
npm install retro-futuristic-ui
# 在你的 React 项目里
import { Button, Card } from 'retro-futuristic-ui';
import 'retro-futuristic-ui/themes/alien.css';
```

### 用法 2：克隆下来当灵感库

```bash
git clone https://github.com/Imetomi/retro-futuristic-ui-design.git
cd retro-futuristic-ui-design
pnpm install
pnpm dev  # 打开 localhost 看效果
```

### 用法 3：Fork + 改造

- 改字体（换 Orbitron → Audiowide 等）
- 改配色（换主题）
- 加自定义组件
- 发 PR 回 Imetomi

## 关键学习要点

1. **完整的 React + TypeScript 项目**——是学现代前端栈的好样本
2. **完整设计 tokens**——这是建立 UI 一致性的关键
3. **多种主题切换**——演示 Design System 的灵活性
4. **Storybook 文档**——组件库标配
5. **真实组件级代码**——不是样式演示，是生产可用
