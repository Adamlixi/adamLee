## 1. dotnet/Comet — C# MVU UI Toolkit ⚠️ archived

| 字段 | 值 |
|---|---|
| 仓库 | github.com/dotnet/Comet |
| Stars | 1,673 |
| Forks | 124 |
| License | MIT |
| 语言 | C# |
| 状态 | ⚠️ **archived**（不再维护） |
| 描述 | "Comet is an MVU UIToolkit written in C#" |

### 背景
- 2020 年由 **jamesclancey** 创建
- "Comet" = 工具包名（无特殊含义）
- MVU = Model-View-Update 架构（类似 Elm / SwiftUI）
- 设计受 Xamarin / MAUI 启发
- 曾短暂隶属于 dotnet 组织（dotnet org GitHub）

### 为什么 archived？
- MVU 在 C# 生态未成主流
- MAUI 2022 发布后，dotnet 团队推 MAUI
- 作者转去 MAUI / Avalonia / Uno Platform
- 仓库保留但不接受 PR

### 替代品
- **MAUI**（.NET 官方）
- **Avalonia**（跨平台）
- **Uno Platform**（WebAssembly 友好）

## 2. rpamis/comet — Agent Skill Harness ✨ 活跃

| 字段 | 值 |
|---|---|
| 仓库 | github.com/rpamis/comet |
| Stars | 2,357 |
| Forks | 234 |
| License | MIT |
| 语言 | TypeScript + Python |
| 状态 | ✅ 活跃 |
| 描述 | "Comet: agent skill harness for turning ideas into evaluated workflows" |

### 核心能力
- 把"想法"转为"评估过的 workflow"
- agent skill 抽象层
- 内置 evaluation / testing
- 跨语言（TS + Python）

### 用法
```typescript
// 定义 skill
const mySkill = defineSkill({
  name: "summarize",
  input: "string",
  output: "string",
  evaluate: (input, output) => ({ quality: 0.95 })
});
```

### 适合谁？
- 想要"评估驱动"的 agent 开发
- 不喜欢纯 prompt 工程
- 喜欢 type-safe 工作流

## 3. CometChat — 实时聊天 SDK（独立公司）

| 字段 | 值 |
|---|---|
| 公司 | CometChat Inc. |
| 官网 | cometchat.com |
| 创立 | 2017，印度班加罗尔 |
| GitHub | github.com/cometchat |
| License | 部分开源（skills 50★）+ 闭源核心 |
| 描述 | "chat / voice / video SDK for apps" |

### 产品
- Chat SDK（消息）
- Voice / Video SDK
- AI 客服
- Extensions（开源 skills）

### 竞品
- **SendBird**（韩国，闭源）
- **Stream**（美国，闭源）
- **Twilio Conversations**
- **PubNub**

### 开源仓库
- **cometchat/cometchat-skills** (50★) - 技能扩展
- 核心 SDK **闭源**

### 何时选 CometChat？
- 你要给 app 加聊天 / 语音 / 视频
- 需要 SDK（不是自己写 WebSocket）
- 不想用 Firebase
