> 适合：已学完 UI 基础（能用 Figma / 理解设计原则）想专项学复古未来风格的设计师
> 目标：30 天交付一个**完整的复古未来 UI Kit / 一个完整产品页**

## 前置准备

```
✓ Figma 能流畅使用（含 Variants、Auto Layout、Components）
✓ 至少完成 5 个 UI 组件设计练习
✓ 选定 1 个子流派（推荐 Cassette Futurism 入门）
✓ 安装 Orbitron + VT323 + Audiowide 字体
```

## 4 周路径（每周 6 天，每天 1-2 小时）

### 第 1 周：视觉语言解构 + 子流派确定

| Day | 任务 | 输出 |
|-----|------|------|
| D1 | 选定子流派（参考 Subgenres）+ 收集 20 张参考图 | 1 张 Pinterest/Figma 情绪板 |
| D2 | 拆解 5 个参考图：色板 / 字体 / 间距 / 元素层级 | 1 张 1px 栅格拆解 + 配色清单 |
| D3 | 提纯配色变量（参考 Color-Palettes） | 1 个 Figma Variables 配色系统 |
| D4 | 选定 3 款字体 + 4 款粗细 + 字号阶梯 | 1 张 Typography Style |
| D5 | 用 Figma 画 4 个基础组件：按钮 / 输入框 / Card / 标签 | 4 个 Components |
| D6 | 拼装 1 个完整组件库 + 命名规范 | 1 个 Figma 库 |

### 第 2 周：核心元素开发

| Day | 任务 | 输出 |
|-----|------|------|
| D7-D8 | 实现 CRT 扫描线 + 辉光 + 厚按钮 三个 CSS 效果（HTML 复刻） | 1 个 CodePen / HTML demo |
| D9-D10 | 实现 Glitch + 终端风 + 网格背景（参考 CSS 笔记） | 1 个完整 demo 页面 |
| D11 | 整合 6 个 CSS 效果到 1 个组件库 demo 页 | 1 个 demo 网站 |
| D12-D13 | 用 Figma 设计 20 个复古未来风格组件（按钮状态 / 卡片 / 表单 / 弹窗 / Tabs） | 20 个 Components |
| D14 | 整理成 1 个 Figma UI Kit + 发布到 Figma Community | 1 个公开 UI Kit |

### 第 3 周：完整产品页设计

| Day | 任务 | 输出 |
|-----|------|------|
| D15 | 选 1 个产品（推荐选择：开发者工具 / 音乐 / 游戏），写产品说明 | 1 段 PRD |
| D16 | 信息架构 + 用户流（sitemap） | 1 张流程图 |
| D17-D18 | 设计 3 个主页面（首页 / 仪表盘 / 设置页） | 3 张高保真原型 |
| D19 | 设计响应式（手机 + 平板 + 桌面）| 3 种断点适配 |
| D20-D21 | 设计 2 个边缘页面（404 / 空状态 / 加载状态） | 2 张扩展页 |

### 第 4 周：交互动效 + 文档 + 发布

| Day | 任务 | 输出 |
|-----|------|------|
| D22-D23 | 设计 10 个微交互（按钮按下 / 弹窗出现 / Glitch 标题 / LED pulse） | 10 个 AE/Rive 动画 |
| D24 | 用 Framer Motion 实现到 web 版本 | 1 个可交互 demo |
| D25-D26 | 整理成 1 个 case study + 撰写设计思路 | 1 篇 Medium 文章 |
| D27 | 拍 5 张 best screens（高质量 visual） | 5 张作品集图 |
| D28-D29 | 录 30 秒产品演示视频 | 1 个视频 |
| D30 | 发布（Dribbble + Behance + Reddit r/UI_Design + r/RetroFuturism）| 1 个公开作品 |

## 关键资源（每天会用到）

| Day | 用到笔记 |
|-----|---------|
| 1-6 | Subgenres / Color-Palettes / Fonts |
| 7-14 | CSS-Retro-Effects / Resource-Imetomi |
| 15-21 | Cassette-Futurism 配色 + 元素 |
| 22-30 | Resource-Community 发布渠道 |

## 验收标准

✅ **30 天后你将有**：
- 1 个公开 Figma UI Kit（含 30+ 组件）
- 1 个完整产品页设计（4-5 屏高保真 + 响应式）
- 1 个可交互 web demo
- 1 篇 Medium 案例文章
- 1 个 30 秒演示视频
- 1 套发布到 Dribbble + Behance + Reddit 的材料

## 🎯 加分项

- 录一个 10 分钟 Loom 视频讲设计思路 → 放 YouTube
- 把 web demo 部署到 Vercel/Netlify → 个人作品集
- 给 Imetomi 发 PR 加新组件 → 开源贡献
- 在 X / Threads 上发拆解 → 个人 IP
