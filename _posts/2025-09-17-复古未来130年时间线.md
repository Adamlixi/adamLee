## 时间线

| 年代 | 关键事件 | 主子流派 | 视觉语言 |
|------|----------|---------|----------|
| **1880-1910** | 凡尔纳《海底两万里》/ 早期科幻 | Steampunk | 黄铜、齿轮、蒸汽 |
| **1920-1940** | 大萧条 / Art Deco 高峰 / 二战 | Decopunk / Dieselpunk | 几何 / 流线型 / 战争机械 |
| **1950** | 美苏太空竞赛 / 原子能狂热 | Raygun Gothic / Atompunk | 火箭 / 原子 / 鲜艳 / 梦幻 |
| **1960** | 阿波罗登月 / 太空时代 / 邦德电影 | Space Age | 银色 / 流线型 / 太空 |
| **1970** | 朋克运动 / 《星球大战》/ VHS 普及 | Proto-Cassette / Cyberpunk | 模拟 / 廉价 / 粗糙 |
| **1980** | 《银翼杀手》/ 任天堂 NES / MTV | **Cassette Futurism** + Cyberpunk | 厚按钮 / CRT / 摩斯电码 |
| **1990** | 《黑客》动画 / 《攻壳机动队》/ 万维网 | Cyberpunk 高峰 | 霓虹 / HUD / 全息 |
| **2000** | 千年虫恐慌 / iMac G3 / Matrix | **Y2K / Cybercore** | 透明塑料 / 渐变 / 太空时代回潮 |
| **2005-2013** | iOS 早期 / Win Vista Aero / 全球化清新 | **Frutiger Aero** | 天空 / 草地 / 气泡 / 玻璃感 |
| **2016** | Stranger Things / 50% 美国怀旧 | 复古未来回潮起点 | 1980s 致敬 |
| **2020** | 疫情 / 加密潮 / AI 突破 | Cyberpunk + Y2K + Cassette 同台 | 多元 |
| **2024** | Fallout TV (Amazon) / Alien TV (FX) | **Cassette Futurism** 成主流 | 模拟时代回归 |
| **2025** | Severance / Devs / Dark Matter | Y2K + Cyberpunk 复苏 | 通用复古未来 |
| **2026** | 通用潮（多元化） | Y2K + Cyberpunk + Cassette 三足鼎立 | 三方并存 |

## 每个时代的"未来想象"特征

```
1880-1910  Victorian   = 飞艇、齿轮钟、蒸汽机械
1920-1940  Decopunk    = 摩天楼、Art Deco、流线型
1950       Atomic      = 飞碟、原子能、机器人管家
1960       Space Age   = 太空站、月球殖民地、银色
1970-80s   Cassette    = 模拟按钮、VHS 噪点、监控仪表
1980-90s   Cyberpunk   = 霓虹、雨、全息广告
1990s      Y2K         = 透明塑料、彩虹渐变、太空感
2005-13    Frutiger    = iOS 美学、玻璃、田园
```

## 几个关键事实

- **130 年光景**——复古未来不是"怀旧"，是"持续的"——每隔 10-15 年，子流派都会换一批
- **Cassette Futurism 的复兴**早在 2010s 就开始了，2024 后随着 Fallout TV / Alien TV / Severance 三连击才进入主流
- **Y2K 在 2024 已经回潮**（Gucci / Versace / Miu Miu 系列）——2026 仍是高审美
- **Frutiger Aero 是 2024-2025 现象级复兴**——微软 Zune / iOS 6 截图集 / "Windows Bliss"壁纸文化复古
- **2026 三大主流** = Y2K + Cyberpunk + Cassette Futurism
