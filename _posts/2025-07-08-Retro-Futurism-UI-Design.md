## ⚡ 关键结论（5 条）

1. **学习 UI 设计 ≠ 学习复古未来风格**——前者是**通用基础**（设计原则 + 工具 + 实战 + 作品集），后者是**垂直风格分支**（特定历史时期的视觉语言）。正确顺序是**先打 UI 基础，再深入复古未来**。
2. **UI 设计自学的 ROI 比想象的高**：Google AI Overview 2026 推荐路径是 **9 个月从零到作品集**；**Refactoring UI + Laws of UX + Material Design 3** 大免费资源够入门；Figma 是事实标准。
3. **复古未来 = 130 年历史的伞概念**，下面 **9 大子流派**——直接说"做复古未来" = 无意义，**必须先选定子流派**，否则会做出四不像。
4. **Cassette Futurism 是 2026 复古未来 UI 爆款子流派**——Imetomi 开源 React 组件在 Reddit r/RetroFuturism 三连发 292 票；代表 Alien / Blade Runner / Halo / Stranger Things / Severance / Fallout TV。
5. **复古未来字体生态已成熟**：Google Fonts 至少有 8 款免费可商用字体（Orbitron / Audiowide / VT323 / Poiret One / Monoton / Michroma / Oxanium / Rajdhani）专门为这个风格设计，**不用花一分钱做字体授权**。
