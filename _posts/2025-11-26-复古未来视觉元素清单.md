## Cassette Futurism 元素（最重要，2026 主流）

### 必须有的元素（5 个核心）

1. **厚按钮** = box-shadow `inset 0 1px 0 rgba(255,255,255,0.2), 0 4px 0 #2a2a2a`
2. **CRT 扫描线** = 整个页面覆盖 2px 双色细线（透明度 30%）
3. **文字辉光** = text-shadow 多层叠加（0 0 5px + 0 0 10px + 0 0 20px）
4. **拉丝金属边框** = linear-gradient 制造金属感
5. **LED 指示灯** = 圆点 + pulse 动画（绿/橙/红）

### 加分元素（5 个进阶）

6. **CRT 边界曲率**（早期是直的，后变曲 — 看你的设定年代）
7. **边角锯齿 / 切角**（45° chamfered corners）
8. **半透仪表盘**（SVG 半圆弧 + 旋转指针）
9. **脉冲动画**（呼吸效果，关键状态用）
10. **复古噪点 / 颗粒**（CSS filter + SVG filter）

## Cyberpunk 元素

### 必须有的元素（5 个核心）

1. **霓虹边框** = box-shadow 多层 5px / 10px / 20px 渐变色
2. **HUD 网格** = 透明 panel + 半透明背景
3. **雨纹纹理** = SVG 噪声 + 流体动画（可选）
4. **扫描线**（斜 45°，更密集）
5. **故障 / Glitch 效果** = clip-path 文字分层 + RGB 偏移

### 加分元素

6. 全息投影视觉（holographic）
7. 终端 / 命令行文本
8. 城市天际线剪影
9. 数据流（如 Matrix 字符雨）
10. 反乌托邦剪影 + Logo

## Y2K 元素

### 必须有的元素（5 个核心）

1. **透明塑料感** = backdrop-filter blur() + 半透明 rgba 背景
2. **双色渐变** = linear-gradient(45deg, #00ffff, #ff00ff) 经典
3. **亮银 / 香槟金** = 金属感点缀
4. **闪光/火花图标** = SVG 星星 + 反光
5. **巨型标题字** = display 字号 96px+（Logo / 首屏）

### 加分元素

6. 玻璃球 / 水珠 / 气泡（cute 元素）
7. 笑脸 logo（2000 年代典型）
8. 心形 / 星形装饰
9. 像素块（8-bit 风插画）
10. PSP / iPod 式 UI 控件

## Frutiger Aero 元素

### 必须有的元素（5 个核心）

1. **玻璃质感** = 半透 panel + 模糊背景（backdrop-filter blur）
2. **蓝天 + 草地背景图**（Aero 标志性）
3. **气泡 / 球体** = 充满光感
4. **柔和阴影** = 蓝色调阴影
5. **鲜艳但清新的颜色**（绿松石 + 橙 + 蓝）

### 加分元素

6. 大玻璃感 Logo
7. 自然元素照片叠加
8. 3D 立体感
9. 光线 / 阳光感
10. 花朵 / 阳光光晕

## Space Age 元素

### 必须有的元素（5 个核心）

1. **星空背景**
2. **火箭 / 卫星图标**
3. **轨道曲线**
4. **太空感渐变**
5. **宇航员剪影**

### 加分元素

6. 雷达扫描动画
7. 行星环
8. 太空船飞行轨迹
9. Mission Control 仪表
10. 漂浮星点

## Decopunk 元素

### 核心

1. **几何对称**（楼梯 / 扇形 / 星形）
2. **黑金配色**
3. **奢华装饰图案**
4. **流线型**
5. **Art Deco 字体**

## Steampunk 元素

### 核心

1. **黄铜齿轮**
2. **皮革缝线**
3. **木板纹理**
4. **煤油灯**
5. **铜管 / 蒸汽喷口**

## 🎯 选择元素的关键问题

```
1. 你的产品类型？
   软件产品 → Cassette / Cyberpunk
   硬件产品 → Steampunk / Space Age
   消费品品牌 → Y2K / Frutiger Aero
   奢侈品 / 高端 → Decopunk / Cyberpunk
   大众产品 → Frutiger Aero / Y2K

2. 你想突出的元素层级？
   主元素（5 个必做） = 决定风格
   加分元素（5 个选做） = 体现深度
   装饰元素（自由发挥）= 体现品牌特色

3. 你的实现难度？
   纯 CSS / Figma  = Cassette / Cyberpunk / Frutiger Aero / Y2K
   需要 SVG / 动画  = Space Age / Cyberpunk
   需要 3D         = Space Age / Steampunk
```
