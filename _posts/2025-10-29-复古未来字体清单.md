## 字体搭配公式

### 公式 1：Cassette Futurism

```
主标题：    Orbitron 或 Audiowide（粗体、大字号）
副标题：    Michroma 或 Oxanium（中等字号、全大写）
正文：      Share Tech Mono 或 Rajdhani（14-16px）
代码 / 数据：  VT323（占位 / 终端）
```

### 公式 2：Y2K

```
主标题：    Monoton（独占一行）
副标题：    Poiret One
正文：      Quicksand 或 Nunito
点缀：      Press Start 2P（保留用像素感）
```

### 公式 3：Frutiger Aero

```
主标题：    Josefin Sans（艺术字）
副标题：    Quicksand
正文：      Nunito（圆润友好）
点缀：      Cormorant Garamond（加入优雅衬线对比）
```

## 决策树：选什么字体？

```
你是哪个子流派？
├── Cassette Futurism → Orbitron / Audiowide / VT323
├── Cyberpunk        → Share Tech Mono / VT323 / Orbitron
├── Y2K              → Poiret One / Monoton / Bungee
├── Frutiger Aero    → Quicksand / Nunito / Josefin Sans
├── Decopunk         → Josefin Sans / Poiret One
├── Space Age        → Orbitron / Michroma
├── Atompunk         → Michroma / Major Mono Display
└── Raygun Gothic    → Monoton / Audiowide

你需要传达什么调性？
├── 神秘高级        → Orbitron + Share Tech Mono
├── 友好清新        → Quicksand + Nunito
├── 街头潮流        → Bungee + Press Start 2P
└── 复古优雅        → Poiret One + Cormorant Garamond
```
