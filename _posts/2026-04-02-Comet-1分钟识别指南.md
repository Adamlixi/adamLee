| 问题 | Perplexity Comet | Comet ML / Opik | dotnet/Comet | rpamis/comet | CometChat | getcomet.net |
|---|---|---|---|---|---|---|
| 1. 涉及 AI/LLM？ | ✅ AI 浏览器 | ✅ ML/LLM 观测 | ❌ C# UI 库 | ✅ agent harness | ❌ 聊天 SDK | ❌ Reddit 替代 |
| 2. 在浏览器跑？ | ✅ 浏览器 | ❌ SDK/Web | ❌ 移动/桌面 | ❌ Node/Python | ❌ 应用内聊天 | ❌ Web 论坛 |
| 3. 公司名带 Comet？ | ❌（Perplexity） | ✅（comet.com） | ❌（Microsoft） | ❌（rpamis） | ✅（CometChat） | ✅（独立） |
| 4. GitHub 有同源仓库？ | ❌ 完全闭源 | ✅ 71 个公开 | ✅ 已 archived | ✅ 活跃 | ✅ 闭源核心 | ⚠️ site 已下线 |

## 一行回答（按用户最可能问的顺序）

### Q1: "我想自己跑一个 Perplexity Comet"
```
Perplexity Comet 是闭源（系统提示词都被泄露过）。
最强开源替代是 BrowserOS（12k stars, AGPL, YC S24）。
```
→ 看 04-Perplexity-Comet-Alternatives

### Q2: "我想用 Comet 做 ML 实验管理"
```
comet.com 是闭源 SaaS，但 Python SDK 完全开源（pip install comet_ml）。
Self-host 需要谈 enterprise 或用 terraform-aws-comet-stsaas。
```
→ 看 05-Comet-ML-Company-Products

### Q3: "我想做 LLM 观测 / eval"
```
直接用 Opik（comet-ml/opik，Apache 2.0，20,685 stars）。
Docker compose 5 分钟跑起来，自带 UI。
```
→ 看 06-Opik-Deep-Dive

### Q4: "我想统一调用 100+ LLM API"
```
用 litellm（53,972 stars，已被 Comet 公司收购）。
OpenAI 格式直接调 Bedrock / Azure / Anthropic / Vertex / Cohere。
```
→ 看 05-Comet-ML-Company-Products

### Q5: "我想做 C# UI"
```
dotnet/Comet，MIT 协议，1,673 stars，但已 archived。
替代品：MAUI / Avalonia / Uno Platform。
```

### Q6: "我想做 agent skill / workflow"
```
rpamis/comet，2,357 stars，MIT，"agent skill harness for turning ideas into evaluated workflows"。
TypeScript + Python。
```

### Q7: "我想加聊天 / 语音 / 视频 到我的 app"
```
CometChat（cometchat.com），不同公司，闭源核心 + 开源 skills。
竞品：SendBird / Stream / Twilio Conversations。
```

## 🔍 命名冲突史（为什么这么多 Comet？）

| 命名年份 | 项目 | 含义 |
|---|---|---|
| ~2010s | dotnet/Comet | "Comet" = C# MVU 工具包 |
| ~2017 | Comet ML（comet.com） | "Comet" = 实验管理 |
| ~2020 | CometChat（cometchat.com） | "Comet" = 聊天 SDK |
| 2024-07 | Perplexity Comet | "Comet" = AI 浏览器（Perplexity 押注的下一代产品） |
| 2025+ | rpamis/comet | "Comet" = agent skill |
| 2026 | Comet Assistant（泄露） | "Comet" = Perplexity 浏览器内置助手 |

**巧合或趋势？**
- "Comet" 在科技圈 = "创新 + 快 + 接近 AI 前沿"的隐喻
- Perplexity 选 Comet 是因为 Comet = 彗星 = "快速穿越宇宙" = 隐喻他们的 AI 搜索速度
- Comet ML 选 Comet = 彗星轨迹 = "看到实验全貌"

## 🧪 实测识别（如果你想自己确认）

```bash
# 1. 如果项目带 "Perplexity" 字样 → Perplexity Comet（闭源）
# 2. 如果项目带 "comet.com" / "comet-ml" → Comet ML 公司（Opik 开源）
# 3. 如果项目是 C# / .NET → dotnet/Comet（archived）
# 4. 如果项目是 TypeScript + agent → 可能是 rpamis/comet
# 5. 如果项目是聊天 SDK → CometChat（不同公司）
```
