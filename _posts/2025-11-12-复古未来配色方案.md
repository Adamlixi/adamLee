## 1. Cassette Futurism 配色

```css
/* 主背景 */
--bg-base:    #0a0a0a;   /* 深黑底（不是纯黑，略带层次） */
--bg-panel:   #1a1a1a;   /* 面板背景 */
--bg-elevated:#2a2a2a;   /* 上浮元素 */

/* 边框 / 拉丝金属 */
--border:     #3a3a3a;
--brushed:    linear-gradient(180deg, #4a4a4a 0%, #2a2a2a 50%, #4a4a4a 100%);

/* 文字 */
--text-primary:    #e0e0e0;   /* 主文字（冷白月光） */
--text-secondary:  #a0a0a0;
--text-tertiary:   #6a6a6a;

/* 主操作 */
--accent-primary:  #ff6b00;   /* 经典橙（Alien 风） */
--accent-success:  #00ff41;   /* 终端绿 / LED 绿 */
--accent-info:     #0099ff;   /* 等离子蓝 */
--accent-warning:  #ffcc00;   /* 信号黄 */
--accent-danger:   #ff3333;   /* 事故红 */

/* 文字辉光 */
--glow-primary:    0 0 10px #ff6b00, 0 0 20px #ff6b00;
--glow-success:    0 0 10px #00ff41, 0 0 20px #00ff41;
```

**代表作品**：Alien (1979)、Blade Runner、Halo、Stranger Things、Fallout TV

## 通用配色规则

### 60-30-10 法则（每个子流派都适用）

```
60%  主背景色（黑 / 浅色背景）
30%  次色（强调 / 面板）
10%  强调色（行动点 / 按钮）
```

### 复古未来配色一致性秘诀

1. **3 主色原则**——不超出 3 种主色 + 3 状态色（success/warning/danger）
2. **辉光配色**——科技感配色必须配 glow（透明度 30-50% 的 color 重叠阴影）
3. **对比度**——主文字 vs 背景对比度 ≥ 4.5:1（WCAG AA 标准）
4. **状态色**——黄/绿/红是通用的"系统状态"信号（终端风）
