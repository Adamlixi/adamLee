> 综合 1-3 章 + Cassette + AI 后的具体行动清单

## 第 1-7 天（启动）

1. **安装字体**：下载并安装 7 款 Cassette Futurism 字体
   - Orbitron / Audiowide / VT323 三个先下
   - 在 Figma 里激活即可用

2. **选定子流派**：参考 6 因素决策表
   - 99% 概率选 **Cassette Futurism**（最适合入门 + 主流）
   - 真喜欢 Cyberpunk → 直接选 Cyberpunk
   - 真喜欢 Y2K → 直接选 Y2K

3. **建情绪板**：Pinterest 创建 1 个 board 收藏 20 张参考图
   - 包括：Alien / Stranger Things / Severance / Fallout TV 等影视截图
   - 也包括：Dribbble "cassette futurism" 搜索 Top 20
   - 也包括：图标的单元素参考（按钮 / 输入框 / 仪表盘）

4. **测试工具流**：用 笔记 11-CSS-Retro-Effects 的 6 大 CSS 片段
   - 复制粘贴到 1 个 HTML 文件
   - 浏览器打开看效果
   - 替换颜色尝试 1-2 次

## 第 8-30 天（基础）

5. **Figma UI Kit v1**：参考 笔记 7 + 笔记 12 的 Day 5-6 + D22-23
   - 至少做 6 个核心组件：Button / Input / Card / Modal / Tabs / Tooltip
   - 配色参考 笔记 9
   - 完成 1 个 Figma Library

6. **代码复刻 v1**：用 Imetomi 项目克隆下来运行
   ```bash
   git clone https://github.com/Imetomi/retro-futuristic-ui-design.git
   cd retro-futuristic-ui-design && pnpm install && pnpm dev
   ```
   - 改 1 个组件（比如按钮颜色），确认你能影响代码
   - 在本地加 1 个新组件（最简单：1 个状态指示灯）

7. **完成 1 个完整产品**：参考 Day 15-21
   - 推荐：1 个开发者工具（CLI dashboard / Cron 监控 / API 监控）
   - 必须是**真实能用的**，不是演示
   - 设计 + 完整代码 + 部署到 Vercel

## 第 31-90 天（深入）

8. **发布到社区**：参考 笔记 14
   - Dribbble 1 个 shot（高保真截图）
   - Behance 1 个 case study（讲故事）
   - Reddit r/RetroFuturism / r/web_design 发项目链接
   - X / Threads 短帖带截图 + 视频

9. **投资组合化**：把 3 个项目（30 天 / 60 天 / 90 天）整理到 1 个个人网站
   - 域名建议：xxx.retrofuturism.studio
   - 或：xxx.design
   - Vercel/Netlify 部署，免费

10. **拓展 1 个新子流派**：复盘"复古未来"基础后，再选 1 个相邻子流派
    - Cassette + Cyberpunk 是紧邻的
    - Y2K + Cybercore 是紧邻的
    - Space Age + Atompunk 是紧邻的
    - 选 1 个相邻 = 复用 50% 知识

## 第 91+ 天（长期）

11. **持续学习节奏**（建议每周 2-3 小时）：
    - 周一：刷 1 小时 Dribbble + Behance 找新参考
    - 周三：看 1 部电影（按子流派清单选）
    - 周六：做 1 个 Side Project（1-4 小时）

12. **AI 工具探索**：参考 笔记 15
    - 如果你愿意投入半年以上 → 试试做 1 个"截图 → UI Kit"小工具
    - 如果你只是想消费 → 留意新产品发布，AI 工具赛道变化很快

## 🎁 额外建议

- **找到 1 个设计导师**：在设计师 Discord / Dribbble / Reddit 关注 1-3 个复古未来设计大牛，长期跟学
- **加 1 个设计师社区**：Discord 上有很多「RetroFuturism」「Cyberpunk UI」「Vaporwave」设计服务器
- **准备 6 个月才有心流**：复刻 Alien（1979）一部电影的视觉，需要至少 6 个月深耕
