## 公司战略：开源 + SaaS 双引擎

```
开源层（吸量 + 生态）
  Opik (LLM obs) ──┐
  litellm (LLM gw) ├──► 开发者入口
  comet_ml SDK ───┘

SaaS 层（变现 + 锁定）
  Experiment Tracking ───┐
  Model Registry ────────┤
  Production Monitoring ─┤──► 企业付费
  Team Collaboration ────┘
```

**关键洞察**：
1. **开源产品 = 流量入口**：Opik 20k stars + litellm 53k stars 都是开发者市场品牌建设
2. **SaaS 产品 = 现金流**：传统 ML 实验管理是付费主力
3. **战略转折**：2023 年之前是"实验追踪"公司，2023 年之后变成"AI/LLM 可观测性"公司
4. **litellm 收购**：2024 年收购 BerriAI 是公司向 AI gateway 转型的关键一步
5. **2026 现状**：公司 70% 估值应该建立在 Opik + litellm 生态上，传统 ML 业务退居二线

## Self-host 现状

### Opik 自部署（最容易）
- **Docker compose**：官方提供 `opik/deployment` 仓库，5 分钟跑起来
- **K8s**：Helm chart 可用
- **资源**：2-4 GB RAM + PostgreSQL + ClickHouse
- **文档**：https://www.comet.com/docs/opik/

### Comet ML SaaS Self-host（仅企业）
- **terraform-aws-comet-stsaas** (2 stars)：AWS Terraform 模块
- **Single-Tenant SaaS** 部署模式
- **需要企业合同**
- 仓库虽开源（仅 IaC），但服务本体闭源

### litellm 自部署
- **完全开源**：Apache 2.0 + Rust 核心
- **CLI**: `litellm --proxy` 起一个 proxy server
- **Docker**: 官方镜像可用
- **K8s**: Helm chart 社区维护
