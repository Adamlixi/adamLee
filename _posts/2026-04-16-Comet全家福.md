## 命名冲突的根因

**3 个独立公司选了 Comet 这个名字**：
1. **Perplexity AI**（2024）— 选 Comet 是隐喻"彗星快速穿越" = AI 搜索速度
2. **Comet ML Inc.**（2017，YC W17）— 选 Comet 是隐喻"实验轨迹可追踪"
3. **CometChat Inc.**（2017，印度）— 选 Comet 是隐喻"快速实时的聊天"

**3 个开源作者独立选了 Comet**：
1. **dotnet/Comet**（Clancey, ~2020）— Comet = "C# 工具包名"
2. **rpamis/comet**（2025）— Comet = "agent 工具"
3. **getcomet.net**（2025）— Comet = "Reddit 替代品"

## 4 个最容易被搞混的 Comet 对照

### Perplexity Comet vs Comet ML
- **Perplexity Comet** = 消费级 AI 浏览器，给普通人用
- **Comet ML** = 企业级 ML 实验平台，给数据科学家用
- **完全无关**，只是同名

### Comet ML vs Comet Opik
- **Comet ML** = 传统 ML（PyTorch/TensorFlow）实验管理
- **Comet Opik** = LLM 应用 observability（Comet ML 2023 推出的新产品）
- **同公司**，但产品线不同

### comet-ml/litellm vs BerriAI/litellm
- **BerriAI/litellm** = 原项目（53,972 stars）
- **comet-ml/litellm** = Comet 收购后建立的镜像（2 stars）
- **同一项目**，新仓库只是品牌镜像

### dotnet/Comet vs Comet ML
- **dotnet/Comet** = C# UI Toolkit
- **Comet ML** = Python ML 平台
- **完全无关**
