> "空白信号" = 在 HN / Reddit / Product Hunt / X 上**没有发现任何成熟产品**，但用户反复讨论要这样的工具。

## 1. 上传电影截图 → 生成完整 Figma UI Kit

**痛点**：
- 设计师看到一张喜欢的复古未来视觉（异形仪表盘 / Severance 工厂 UI / Fallout PipBoy）
- 想直接基于它做自己的产品 UI
- 现在必须手动提取配色 / 字体 / 间距 / 元素 → 慢

**空白度**：🔴 0 个成熟产品
- 现有：Figma AI、Visily、Magician Design — 不专门做复古未来
- 现有：[AI Image to Figma](https://www.figma.com/community/plugin/...) — 通用但不专

**MVP 思路**：
1. 用户上传电影截图（或参考图）
2. AI 识别子流派（cassette / cyberpunk / y2k）
3. 输出 Figma Kit：配色 tokens + 字体 + 基础组件 + 首页模板
4. 用户可以微调后再导出

**商业模式**：
- B2B 设计订阅（设计师 / 中小团队）$19/月
- B2C 单次 $5-10 / 包年 $49
- API：批量给企业客户 $0.5/次

## 2. 复古未来字体训练数据集 + 模型

**痛点**：
- 现在每张图都要手动找对应子流派字体
- 设计师常混淆几个子流派
- 没有 1 个权威数据集说"这是 Cassette Futurism 字体"

**空白度**：🔴 0 个公开数据集
- 现有：Google Fonts 有 16 款免费（散落）
- 现有：HuggingFace 有 OCR / 字体识别，但是通用
- 没有：「Cassette Futurism 字体 → CSS 代码」映射集

**MVP 思路**：
1. 建 1 个标注好的数据集（3000+ 字体样本 × 子流派标签）
2. 训练 1 个 fine-tune 模型识别图像里的字体子流派
3. 输出对应的 Google Fonts CSS import
4. 开源数据集 + 提供 API

**商业模式**：
- 数据集授权给字体公司
- 模型 API 按调用收费
- 开源 = 品牌效应，付费 = API

## 3. AI 复古未来配色 + 字体推荐器

**痛点**：
- 设计师选了子流派但配色 / 字体组合还是手动试错
- 现有工具（Coolors / Adobe Color / Khroma）通用，无垂直

**空白度**：🔴 0 个垂直产品

**MVP 思路**：
- 输入：子流派 + 主题色 + 调性
- 输出：5 套完整配色 + 3 款字体组合 + 主题预览

**竞品对比**：

| 工具 | 是否复古未来垂直 |
|------|---------------|
| Coolors | ❌ 通用 |
| Adobe Color | ❌ 通用 |
| Khroma | ❌ 通用 AI |
| ColorHub | ❌ 通用 |
| Colormind | ❌ 通用 |
| **Retro Color（空白）** | ✅ 复古未来专用 |

## 4. Figma Plugin 一键生成 UI Kit

**痛点**：
- 设计师安装了 Imetomi 但要切到 web 看效果
- 想在 Figma 里直接一键生成复古未来风格的 UI Kit

**空白度**：🟡 有 1 个接近的（[Retro UI Kit Generator](https://www.figma.com/community/...)）但功能浅

**MVP 思路**：
1. 选定子流派 + 配色 + 字体
2. 一键生成 30+ 组件（按钮 / 卡片 / 表单等）
3. 自动应用子流派视觉规则（CRT 扫描线 / 厚按钮等）
4. 一键导出 CSS 代码

## 5. Reference → Figma Converter（参考图转 Figma）

**痛点**：
- 看到一个复古未来的 website / app
- 想快速复刻到 Figma 里学
- 现在必须手画

**空白度**：🔴 0 个复古未来专用（通用版本有）

**MVP 思路**：
1. 输入：URL 或截图
2. AI 分析：颜色 / 字体 / 间距 / 元素层级
3. 输出：Figma 文件（节点结构 + 可编辑）
4. 输出：详细学习笔记（为什么这里用 16px 不是 14px）

## 6. 复古未来 Marketplace / 垂直设计交易

**痛点**：
- 复古未来设计师稀少但有市场（独立游戏 / 影视 / 音乐产品）
- 现在要 1 个复古未来设计师必须到 Fiverr 通用市场搜
- 没有垂直市场

**空白度**：🔴 0 个复古未来专用市场

**MVP 思路**：
1. B2C：设计师市场（按子流派筛选）
2. B2B：设计师招募（订阅式）
3. 资源：UI Kit / 字体 / 模板 出售
4. 学习：课程 + 案例研究

**市场规模估算**：
- 复古未来 UI Kit 单价 $20-100，Dribbble 年活设计师 50 万，估计 1% 感兴趣 = 5000 × 30 美元 × 1 次 = $150k
- 独立游戏市场：复古未来游戏 100+/年 × $500-5000 = $50k-500k
- 影视 UI 设计师稀缺，薪资可能 $80k-150k/年，企业招聘付费

## 🎯 我的推荐：先做 #3 再做 #1

**理由**：
- #3 最容易做（只需训练小型色彩 / 字体模型 / 调用 VLM API）
- #3 直接服务设计师单边市场
- #3 跑通后可作为 #1 的核心功能（一键推荐完整 UI Kit）
- #3 也是 SaaS 友好的订阅模式

**演化路径**：
```
M1-M2：AI 复古未来配色推荐器（最小可行）
  → 接收 100 个种子用户（Reddit / Designer Discord）
M3-M4：AI 复古未来字体推荐器
  → 集成 Google Fonts API 自动生成 CSS
M5-M6：截图 → UI Kit（基于 #3 增功能）
  → 增加年收入 30 万+
M7+：Marketplace（双边网络效应）
  → 已建立用户基础，好启动
```
