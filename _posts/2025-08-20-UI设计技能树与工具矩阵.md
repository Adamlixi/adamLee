## 工具矩阵

| 工具 | 类型 | 价值 | 替代 |
|------|------|------|------|
| **Figma** | UI/UX 设计 | 事实标准，免费，浏览器即用 | Sketch（Mac only）、Adobe XD（停更） |
| **Sketch** | UI 设计 | Mac 老牌标准 | Figma |
| **Adobe Photoshop** | 图像处理 | 海报 / 素材处理 / 复古滤镜 | Affinity Photo |
| **Adobe Illustrator** | 矢量 | Logo / 图标 | Affinity Designer、Inkscape |
| **Procreate** | iPad 绘画 | 手绘 / 草图 / 复古质感 | — |
| **After Effects** | 动效 | 微交互 / Lottie | Rive、Principle |
| **Rive** | 动效 | 实时 + 设计师友好 | AE |
| **Principle** | 原型 | 移动端原型 | Figma 原型 |
| **Blender** | 3D | 免费 3D + 复古未来 3D 元素 | Cinema 4D |
| **Spline** | 3D（Web） | 浏览器 3D，2026 热门 | — |

> **新手 2026 起步套装**：**Figma + Photoshop + Illustrator + Procreate + Blender**。5 个就够用，其他都是这 5 个的延展。

## 第 1 个月必学基础技能（按学习顺序）

1. **Figma 基础**：Frame / Auto Layout / Components / Variants / Variables
2. **字体基础**：字体家族 / 字重 / 字距 / 行高 / 字号阶梯
3. **色彩基础**：HSL / RGB / 色轮 / 互补色 / 类比色 / 中性色阶
4. **网格系统**：12 栏 / 8px 栅格 / baseline grid / 模块化间距
5. **图标设计**：网格图标 / 描边 / 填充 / 视觉重量统一
6. **设计系统**：Component / Design Token / Style / Library
7. **响应式**：断点 / 流式布局 / 移动端适配
8. **可访问性**：对比度 / 字号 / 屏幕阅读器
