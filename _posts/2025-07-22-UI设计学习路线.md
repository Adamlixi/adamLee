## 三条主流路线对比

| 路线 | 时长 | 成本 | 适合谁 | 优势 | 劣势 |
|------|------|------|--------|------|------|
| **完全自学（Online）** | 6-12 月 | ¥0-2,000 | 自律强 / 时间灵活 / 转行者 | 灵活 / 经济 / 可与项目结合 | 反馈缺 / 易走偏 / 作品集难 |
| **集训营（Bootcamp）** | 3-9 月 | ¥10,000-80,000 | 急需转行 / 想系统化 / 在职想跳槽 | 节奏紧 / 有导师 / 有同学社群 | 贵 / 质量参差 / 部分无认证 |
| **学历教育（Design School）** | 2-4 年 | ¥40,000-300,000 | 高中 / 本科申请 / 想拿学位 | 系统 / 有学位 / 校友网络 | 长 / 贵 / 与市场脱节 |

> **2026 主流推荐路线**：以**自学为主**，辅以 1-2 个**短期集训营**（如 Designlab、Springboard、Google UX Design Professional Certificate），最后用 6 个月作品集项目收尾。

## 9 个月自学路线（Google AI Overview 2026）

```
月份     阶段               核心动作                              输出
─────────────────────────────────────────────────────────────────────────
M1       基础               Figma 入门 + UI 基础概念              1 个登陆页克隆
M2-3     设计原则           Refactoring UI + Laws of UX           5 个组件练习
M4-5     工具精通           Figma 进阶 + 设计系统                 1 个完整组件库
M6-7     实战项目           选 2-3 个真实问题做端到端设计          3 个 case study
M8       反馈迭代           Dribbble / Reddit / 设计师社区 critique 修订作品
M9+      作品集 + 求职      Behance + 个人站 + 简历 + Case study   完整作品集
```

## 5 个设计师角色对比

| 角色 | 核心职责 | 工具 | 收入中位数 (US, 2025) |
|------|---------|------|---------------------|
| **UI Designer** | 视觉设计 / 组件 / 设计系统 / 风格 | Figma + Sketch + Adobe XD | $85K |
| **UX Designer** | 用户研究 / 信息架构 / 流程 / 原型 | Figma + Miro + 调研工具 | $98K |
| **Product Designer** | UI+UX+ 业务理解 + 跨团队 | 全栈 | $120K+ |
| **Interaction Designer** | 微交互 / 动效 / 状态过渡 | Principle / Rive / After Effects | $110K |
| **Design Engineer** | 设计 + 代码 | Figma + React/Tailwind | $140K+ |

> **2026 趋势**：市场越来越倾向 **"T 型"人才**——UI 基础扎实 + 至少 1 个垂直领域深耕（品牌 / 动效 / 3D / 数据可视化 / AI 工具设计）。"只会画界面"的纯 UI 设计师需求在缩水，"懂 AI prompt / 能用 AI 工具 / 能做设计系统"的复合需求在涨。
