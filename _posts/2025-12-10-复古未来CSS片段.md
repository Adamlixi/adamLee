## 1. CRT 扫描线效果

```css
.crt-screen {
  position: relative;
}

.crt-screen::before {
  content: "";
  position: absolute;
  inset: 0;
  background: linear-gradient(
    rgba(18, 16, 16, 0) 50%,
    rgba(0, 0, 0, 0.25) 50%
  );
  background-size: 100% 4px;            /* 扫描线间距 */
  pointer-events: none;
  z-index: 10;
}

.crt-screen::after {
  content: "";
  position: absolute;
  inset: 0;
  background: rgba(18, 16, 16, 0.1);
  box-shadow: inset 0 0 100px rgba(0,0,0,0.5);  /* CRT 边缘暗 */
  pointer-events: none;
  z-index: 11;
}

/* 可选：加一层弯曲效果 */
.crt-curved {
  border-radius: 5%;
  transform: perspective(1000px) rotateX(0.5deg);
}
```

**适用**：终端界面、HUD、外星飞船显示屏

## 2. 辉光文字（Neon Glow）

```css
.glow-text {
  color: #00ff41;
  text-shadow:
    0 0 5px    #00ff41,
    0 0 10px   #00ff41,
    0 0 20px   #00ff41,
    0 0 40px   #00ff41;
  font-family: 'Orbitron', sans-serif;
}

.glow-text-orange {
  color: #ff6b00;
  text-shadow:
    0 0 5px    #ff6b00,
    0 0 15px   #ff6b00,
    0 0 30px   #ff6b00;
}

/* 动态辉光 — 关键状态使用，节省 GPU */
@keyframes flicker {
  0%, 19%, 21%, 23%, 25%, 54%, 56%, 100% {
    opacity: 1;
    text-shadow: 0 0 5px #ff6b00, 0 0 15px #ff6b00;
  }
  20%, 24%, 55% {
    opacity: 0.7;
    text-shadow: none;
  }
}
.flicker { animation: flicker 2s infinite; }
```

**适用**：重要数据、警报、系统消息

## 3. 厚按钮按下效果（Physical Button）

```css
.button-thick {
  background: linear-gradient(180deg, #5a5a5a 0%, #3a3a3a 100%);
  border: 2px solid #2a2a2a;
  border-radius: 6px;
  box-shadow:
    inset 0 1px 0 rgba(255,255,255,0.2),  /* 顶部反光 */
    0 4px 0 #1a1a1a,                       /* 底部投影 */
    0 6px 6px rgba(0,0,0,0.3);              /* 整体悬浮 */
  padding: 10px 20px;
  color: #e0e0e0;
  font-family: 'Orbitron', sans-serif;
  font-weight: 700;
  letter-spacing: 1px;
  text-transform: uppercase;
  transition: all 0.1s;
  cursor: pointer;
}

.button-thick:hover {
  background: linear-gradient(180deg, #6a6a6a 0%, #4a4a4a 100%);
}

.button-thick:active {
  box-shadow:
    inset 0 1px 0 rgba(255,255,255,0.1),
    0 0 0 #1a1a1a,                          /* 按下无投影 */
    0 2px 2px rgba(0,0,0,0.3);
  transform: translateY(4px);                /* 按下 4px */
}

/* LED 状态灯组合 */
.button-thick::before {
  content: "";
  display: inline-block;
  width: 8px;
  height: 8px;
  border-radius: 50%;
  background: #00ff41;
  box-shadow: 0 0 5px #00ff41;
  margin-right: 8px;
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0%, 100% { opacity: 1; }
  50%      { opacity: 0.5; }
}
```

**适用**：所有主按钮 — Cassette Futurism 的核心交互元素

## 4. Glitch 文字效果

```css
.glitch {
  position: relative;
  font-family: 'VT323', monospace;
  color: #fff;
}

.glitch::before,
.glitch::after {
  content: attr(data-text);
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  overflow: hidden;
}

.glitch::before {
  color: #ff00ff;
  z-index: -1;
  animation: glitch-anim-1 2s infinite linear alternate-reverse;
}

.glitch::after {
  color: #00ffff;
  z-index: -2;
  animation: glitch-anim-2 3s infinite linear alternate-reverse;
}

@keyframes glitch-anim-1 {
  0%   { clip-path: inset(20% 0 30% 0); transform: translate(-3px, 0); }
  25%  { clip-path: inset(60% 0 10% 0); transform: translate(3px, -1px); }
  50%  { clip-path: inset(40% 0 40% 0); transform: translate(-2px, 1px); }
  75%  { clip-path: inset(10% 0 70% 0); transform: translate(2px, 0); }
  100% { clip-path: inset(50% 0 20% 0); transform: translate(-1px, -1px); }
}

@keyframes glitch-anim-2 {
  0%   { clip-path: inset(30% 0 50% 0); transform: translate(3px, 1px); }
  50%  { clip-path: inset(10% 0 60% 0); transform: translate(-3px, 0); }
  100% { clip-path: inset(50% 0 20% 0); transform: translate(3px, -1px); }
}
```

**适用**：标题 / 大字 / 错误信息（Cyberpunk 必备）

## 5. 终端风输入框

```css
.terminal-input {
  background: #000;
  border: 2px solid #00ff41;
  border-radius: 0;
  padding: 12px 16px;
  color: #00ff41;
  font-family: 'VT323', 'Share Tech Mono', monospace;
  font-size: 18px;
  outline: none;
  width: 100%;
  caret-color: #00ff41;
}

.terminal-input:focus {
  box-shadow:
    0 0 0 1px #00ff41,
    0 0 10px #00ff41;
}

.terminal-input::placeholder {
  color: rgba(0, 255, 65, 0.4);
}

/* 闪烁光标 */
.terminal-prefix::before {
  content: "> ";
  color: #00ff41;
  animation: caret 1s steps(1) infinite;
}

@keyframes caret {
  50% { opacity: 0; }
}

/* 完整终端容器示例 */
.terminal {
  background: #0a0a0a;
  color: #00ff41;
  padding: 20px;
  font-family: 'VT323', monospace;
  border-radius: 6px;
  box-shadow: inset 0 0 50px rgba(0,255,65,0.1);
}
```

**适用**：搜索框 / 命令行 / 单行输入场景

## 6. 网格背景（Reference Grid）

```css
.grid-bg {
  background-color: #0a0a0a;
  background-image:
    /* 主网格线 (64px 间距) */
    linear-gradient(rgba(0,255,65,0.08) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0,255,65,0.08) 1px, transparent 1px),
    /* 次网格线 (16px 间距，更浅) */
    linear-gradient(rgba(0,255,65,0.04) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0,255,65,0.04) 1px, transparent 1px);
  background-size:
    64px 64px,
    64px 64px,
    16px 16px,
    16px 16px;
}

/* 中心十字（瞄准镜效果，可选） */
.grid-bg-with-crosshair {
  position: relative;
}
.grid-bg-with-crosshair::after {
  content: "";
  position: absolute;
  top: 50%; left: 50%;
  width: 80px;
  height: 80px;
  transform: translate(-50%, -50%);
  background:
    linear-gradient(rgba(0,255,65,0.3) 1px, transparent 1px) center/100% 1px no-repeat,
    linear-gradient(90deg, rgba(0,255,65,0.3) 1px, transparent 1px) center/1px 100% no-repeat;
}
```

**适用**：登录页背景、空状态、装饰底层

## ⚠️ 可访问性建议

- 不要依赖单一颜色传达信息（色盲友好）
- 文字辉光不要过大（影响可读性）
- 动画要 `prefers-reduced-motion` 适配
- 扫描线 / 噪点控制在 30% 透明度以内

```css
@media (prefers-reduced-motion: reduce) {
  .flicker, .glitch::before, .glitch::after {
    animation: none;
  }
}
```
