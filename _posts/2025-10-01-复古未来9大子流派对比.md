## 子流派矩阵

## 关键洞察

1. **每个子流派有自己的"年代脊柱"**——不能混搭（例：1970s Cassette + 1960s Space Age = 不伦不类）
2. **2026 三大主流**：Cassette Futurism + Cyberpunk + Y2K，这三个都有**免费 Google Fonts** 支持
3. **小圈子流派**（Steampunk / Dieselpunk / Atompunk）适合**长尾市场**（游戏 / 影视 / 个性化产品）
4. **Frutiger Aero 在 2025-2026 正在复兴**，跟 Windows Vista / iOS 6 怀旧经济有关

## 怎么选子流派（决策表）

```
Q1: 你的产品用户是？
  Z 世代 + 重审美       → Y2K / Cassette Futurism
  千禧一代 + 怀旧        → Frutiger Aero / Y2K
  极客开发者 / 工具人     → Cassette Futurism / Cyberpunk
  艺术 / 影视 / 游戏      → Decopunk / Atompunk
  普通消费者大众市场      → Y2K / Frutiger Aero（柔和）

Q2: 你的产品调性是？
  神秘 / 高端 / 探索      → Cyberpunk / Dieselpunk
  友好 / 温暖 / 自然      → Frutiger Aero / Y2K
  复古 / 80s 模拟        → Cassette Futurism
  太空 / 未来 / 探索      → Space Age / Raygun Gothic
  精致 / 奢华 / 装饰      → Decopunk
```
